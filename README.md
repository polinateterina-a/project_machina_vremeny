# проект "перекрёсток эпох"
Репозиторий проекта "Перекрёсток эпох" — интерактивной веб-платформы, где пользователь может путешествовать по городу во времени, одновременно видя, как менялся облик улиц, и слыша, как они звучали в разные эпохи.
